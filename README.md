# EnviroCast Risk AI

About the Project
EnviroCast Risk AI is a web-based AI tool that predicts future environmental risks
such as extreme heat and air pollution for different locations.
The goal is to help communities prepare for climate-related challenges.

Features
- Predicts future environmental risk levels
- Visualizes climate trends over time
- Simple and user-friendly web interface
- Built with Python and web technologies

How It Works
1. User selects a location
2. The system analyzes historical climate data
3. A prediction model estimates future risk
4. Results are displayed using charts

Tech Stack
- Python
- Flask
- Pandas
- Scikit-learn
- HTML, CSS, JavaScript
- Chart.js

Installation
```bash
pip install -r requirements.txt
python app.py
